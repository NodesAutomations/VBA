## If 

### Code Syntax
```vba
If <Condition> Then <Statement>
```

Else

End if

### Select Case

### Code Syntax
```vba
Select Case Expression
    Case Expression1
	Statement1
    Case Expression2
        Statement2
    Case ExpressionN
        StatementN
End Select 
```
